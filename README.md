# Frontend_Odyssey
The Life of a Developer 🚀
A fun, interactive storytelling website that takes you through the real journey of being a developer — from first "Hello World" to late-night debugging marathons.

✨ What you'll experience
Chapter-based story — 6 stages of developer life
Live interactions — Bug hunting, coffee counter, deploy button
Developer mood meter — Changes based on your actions
Achievement system — Unlock badges as you complete tasks
Fake commit history — See the "real" development process
Dark/Light mode — Smooth theme switching
Progress tracking — Side nav shows your journey progress

🎮 Interactive Features
Feature	What it does
Bug Hunt	Click all 5 bugs to unlock "Bug Hunter"
Coffee Counter	Drink 5+ coffees for "Caffeine Core" badge
Terminal	Run your first code and see it work
Deploy Button	Watch a full deployment animation
Mood Tracker	Changes based on your game progress
Quote Generator	Random developer wisdom
